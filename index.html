Borrowing-asset solver · Base ⇄ KalqiX

Desk simulation

Two settlement strategies replayed against the same real Base swap tape and the same starting capital, so the numbers compare directly: AaveSolver borrows against an Aave position to settle every trade, while LiquiditySolver settles straight out of on-chain inventory mirrored across both venues. Pick a view below.

AaveSolver

Aave-settled solver desk

Settles through Aave collateral/debt on Base, hedged on KalqiX.

LiquiditySolver

Two-sided inventory desk

Settles from plain on-chain inventory, mirrored across Base and KalqiX. No lending market.

Data: data/trades.csv — a snapshot of the trades table, replayed identically by both dashboards. Regenerate it from your own database to update both views at once. See README.md for how to run this locally.
